# gallery-by-react
one photo gallery  project based on react

> --yeoman安装
```bash
npm install -g yo
```

> --generator：项目生成器
```bash
npm install -g generator-react-webpack
```

> --在项目目录里generate
```bash
yo react-webpack gallery-by-react
```
> --项目启动
```bash
npm start
```
